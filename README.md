# weather-report

Previsão do tempo com React + OpenWeatherMap API.

## Stack

- **Lang:** JavaScript
- **Framework:** React + Vite
- **API:** OpenWeatherMap
- **Estilização:** SCSS + Axios

## Como rodar

```bash
cp .env.example .env  # adicionar VITE_API_KEY do OpenWeatherMap
npm install
npm run dev
```
